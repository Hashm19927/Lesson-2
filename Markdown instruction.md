## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить звездочкой (*)ю Например:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить пронумерованные списки, необходимо пункиты просто пронумеровать. Например:
1. Первый пункт
2. Второй пункт
